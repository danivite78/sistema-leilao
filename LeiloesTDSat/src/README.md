Sistema de Gerenciamento de Casa de Leilão

Sobre o Projeto
Este é um sistema desenvolvido para auxiliar a casa de leilão no gerenciamento de suas operações diárias, incluindo cadastro de produtos, controle de lances e gestão de compradores.

Tecnologias Utilizadas
- Java - Linguagem de programação principal
- MySQL - Sistema de gerenciamento de banco de dados
- NetBeans - Ambiente de desenvolvimento integrado (IDE)
- Git/GitHub - Controle de versionamento

Como Executar
1. Importar o banco de dados MySQL
2. Abrir o projeto no NetBeans
3. Configurar as credenciais de conexão com o banco
4. Executar o projeto
